BallDude
========

A third-person OpenGL adventure game.

## Controls

| Keys | Action |
| ---- | ---- |
| WASD or Arrow keys | Move around |
| Escape | End game |
| Right mouse button | Jump |
| Left mouse button | Shoot |
| Space | Emit flatus |
| Q, E | Zoom camera in/out |
| Mouse movement | Move camera |
